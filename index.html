    About

    This is a very simple chrome context menu extension to find selected text on http://mapy.cz/ . It just opens new a tab with the http://mapy.cz/ search result.

    It is for those who are using mapy.cz and find it annoying to copy&paste the address to the mapy.cz search.

    Install

    Get it directly from chrome webstore or grab it from source.

    Support

    Please use either the chrome webstore support or github issues
